# Hello World

This is a minimal bundled skill example.

## When to Use

- Confirm your plugin successfully registered the `skills/` directory
- Check that OpenCode can list and load custom skills from this package

## Quick Check

Ask OpenCode to list available skills, then load `hello-world`.

## Notes

Replace this directory with your real skills as the plugin grows.
